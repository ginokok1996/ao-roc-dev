Wat heb je nodig?

1.        Om met BlueJ te kunnen werken moet je eerst de Java 2 Platform Standard Edition (J2SE) installeren. De voorbeelden uit het boek werken alleen goed met J2SE versie 5.0 of hoger. Op deze cd-rom staat versie 5.0 Update 2. Deze versie heet ook wel, nogal verwarrend, JDK 1.5.

Ga verder met installatie J2SE.

De J2SE bevat de Java Development Kit, waarmee je Java programma's kunt ontwikkelen, en de Java Runtime Environment, waarmee je Java-applicaties en applets kunt uitvoeren.

2.        Daarna moet je BlueJ installeren. BlueJ staat niet op de cd-rom, maar je kunt het gratis downloaden van de BlueJ-website.

3.        Kopieer tot slot de voorbeelden uit het boek van de cd-rom naar je harde schijf.
